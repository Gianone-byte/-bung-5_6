# ScreenStack – Movie Watchlist

## Wie die App gestartet wird:

```bash
npm install
npm run dev -- --open
```

```
src/

│  └─ movies/
│     └─ +page.svelte
└─ lib/
   ├─ components/MovieCard.svelte
   └─ data/movies.json
static/images/
```
